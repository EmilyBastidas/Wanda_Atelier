# Wanda_Atelier

**Wanda Atelier** es una tienda online de bisutería artesanal creada para exhibir y vender piezas únicas diseñadas y elaboradas a mano.

## Objetivo

Desarrollar una plataforma moderna, intuitiva y escalable que permita mostrar el catálogo de productos y, en el futuro, ofrecer una experiencia completa de comercio electrónico mediante una aplicación web y móvil.
